Clone the repository:

```

```

Open the project folder and run:

```
index.html
```

in your browser.

## 🌍 Deployment

This project is deployed using **GitHub Pages**.

Steps used:

1. Upload project to GitHub
2. Go to **Repository Settings**
3. Open **Pages**
4. Select **main branch**
5. Deploy
